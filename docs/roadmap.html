Phase 1
Foundation
v1.0 – v1.2 · Shipped
✓ 10 Health Probes
Process, port, HTTP, config, WebSocket, TUN, memory, CPU, disk, log tail — full gateway observability.
✓ L1/L2/L4 Recovery
Quick restart, targeted repair (6 patterns), human alert cascade with anti-flap and circuit breaker.
✓ 8 Alert Providers
ntfy, Telegram, WhatsApp, Slack, Discord, Email, Pushover, webhook — all out-of-band.
✓ Config Guardian
Dead man's switch, pre-flight validation, two-tier backup (chronological + known-good).
✓ Cross-Platform
macOS (launchd) + Linux (systemd). Auto-detect platform, PID resolution, native probes.
Phase 2
Integration
v1.3 · Shipped
✓ REST API Server DX
18 JSON endpoints for dashboard integration — health, probes, incidents, recovery, config, alerts.
✓ Bot Commands ALERTS
Two-way messaging on Telegram, WhatsApp, Slack, Discord. 8 commands: /health, /status, /incidents, etc.

aegis incidents — list, detail, timeline view. JSONL storage with MTTR calculation.
Phase 3
L3 Recovery
v1.4 · In Progress
Deep Repair Tier RECOVERY
L3 sits between L2 (targeted) and L4 (human). Riskier automated fixes: service reinstall, dependency repair, config migration.
Network Repair RECOVERY
Detect and fix DNS resolution failures, stale routes, TUN interface resets. Inspired by Monit's network checks.
Process Resurrection RECOVERY
If gateway binary is missing or corrupted, attempt re-download and reinstall from known-good source.
Dependency Health RECOVERY
Check node_modules integrity, detect missing native modules, auto-rebuild when possible.
Safe Mode Boot RECOVERY
Start gateway with minimal config (no plugins, default routes) to isolate failures. Like Windows Safe Mode for OpenClaw.
Phase 4
Observability
v1.5 · Next
Prometheus Metrics INFRA
Expose /metrics endpoint in Prometheus format. Probe latencies, health scores, recovery counts, alert delivery times.
Structured Logging DX
JSON log output with severity levels, correlation IDs, and log rotation. Compatible with Loki, ELK, CloudWatch.
Health History PROBES
Time-series storage of probe results. Trend analysis, degradation detection, capacity planning graphs.
SLA Tracking DX
Uptime percentage, SLA compliance reports, availability windows. Inspired by Datadog SLO monitoring.
OpenTelemetry Traces INFRA
Distributed tracing for recovery actions. See the full journey from probe failure to resolution.
Phase 5
Intelligence
v2.0 · Planned
Anomaly Detection AI
Baseline normal behavior, detect deviations before they become failures. Statistical + ML approach inspired by Datadog.
Predictive Alerts AI
Memory leak trajectory, disk fill rate, CPU trend — alert before thresholds are hit. Like PagerDuty's intelligent alerting.
Root Cause Analysis AI
Correlate probe failures, log patterns, and system events to pinpoint root cause. Inspired by Datadog Watchdog.
Runbook Automation RECOVERY
User-defined YAML runbooks for custom recovery procedures. Conditional steps, rollback on failure. Like PagerDuty's runbook automation.
Alert Noise Reduction ALERTS
Group related alerts, suppress duplicates, intelligent escalation. Inspired by PagerDuty's Event Intelligence.
Phase 6
Fleet Management
v2.1 · Planned
Fleet Mode INFRA
Monitor multiple OpenClaw instances from one Aegis — not just local. Aggregate health view across your entire fleet.
Remote Probes INFRA
Health check remote nodes over Tailscale or WireGuard. Monitor any VPS from your machine without opening ports.
Per-Client Dashboard DX
Usage, status, and incidents broken down per client. Know exactly who's healthy and who needs attention.
Deployment Push INFRA
Push updates to satellite nodes from central. Rolling restarts with health verification between each.
Cost Tracking DX
Tokens used per client per day. Track resource consumption across your fleet with per-node breakdowns.
Phase 7
Ecosystem
v2.2 · Planned
Plugin System DX
Custom probes, recovery actions, and alert providers as npm packages. Hook into the probe pipeline with a simple API.
Web Dashboard DX
Standalone web UI with real-time health display, incident timeline, config editor. Built on the REST API.
Terraform Provider INFRA
Manage Aegis config as Infrastructure-as-Code. Declare alert channels, thresholds, and recovery policies in HCL.
GitHub/GitLab Integration DX
Auto-create issues on incidents, link PRs to fixes, deployment status checks. Close the loop from alert to resolution.
Phase 8
Autonomous
v3.0 · Vision
LLM-Powered Diagnosis AI
Feed logs, metrics, and config to an LLM for natural language diagnosis. "Your gateway crashed because the TUN interface lost its route after a DHCP renewal."
Self-Evolving Runbooks AI
Learn from successful recoveries, automatically create new runbook entries. The system gets smarter with every incident.
Chaos Engineering SECURITY
Controlled fault injection to test recovery paths. Kill processes, corrupt configs, fill disks — verify Aegis handles it. Inspired by Netflix Chaos Monkey.
Security Scanning SECURITY
Detect exposed ports, weak permissions, outdated dependencies, CVEs in gateway. Continuous security posture monitoring.
Cross-Service Mesh INFRA
Monitor not just the gateway but its upstream/downstream dependencies. Full service graph health with cascade failure detection.
